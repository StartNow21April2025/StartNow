# AltTabJournal Backend

Spring Boot REST API for the AltTabJournal blogging platform.

## Quick Start

```bash
mvn spring-boot:run
```

## API Documentation
- Swagger UI: http://localhost:8080/swagger-ui.html
- API Docs: http://localhost:8080/v3/api-docs

## Testing
```bash
mvn test
```

## Tech Stack
- Spring Boot 3.2.3
- Java 21
- DynamoDB
- JWT Authentication
- Maven
